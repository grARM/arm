# arms
## 前端模块化开发工具

## 本地全局安装 
* [sudo] tnpm install -g arms

## 使用说明
* arms --help

## 实例
初始化一个工程。
init a project.

``` javascript
$ arms --init 或 arms -i
``` 

## 命令
* arms -i  // 初始化一个项目或js库
* arms -h  // 查看全部命令
* arms -v  // 查看版本
* arms -s  // 以当前目录启动一个本地静态sever
